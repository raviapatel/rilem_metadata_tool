RILEM Metadata Tool
===================

A Gradio-based tool for creating and managing RILEM dataset metadata.

- **Live app**: [HuggingFace Spaces](https://huggingface.co/spaces/raviapatel/rilem-metadata-tool)
- **Documentation**: See [docs/](docs/)
- The app auto-detects local vs HF environment via the `SPACE_ID` env var for `server_name`.
